💰 1. Interest & Time Value of Money
Formula NameFormulaUse When
Simple Interest I = P × i × n
A = P(1 + i·n)		 Interest only on principal; short-term loans
Compound Interest (FV) FV = PV × (1 + i)ⁿ Single amount growing over n periods
Present Value (PV) PV = FV / (1 + i)ⁿ
PV = FV × PVIF(i,n)		 Find today's worth of a future amount
PVIF
PV Interest Factor		 PVIF(i,n) = 1 / (1+i)ⁿ Table lookup for single amount discounting
FVIF
FV Interest Factor		 FVIF(i,n) = (1+i)ⁿ Table lookup for single amount compounding
Effective Interest Rate ieff = (1 + r/m)ᵐ − 1 When compounding is more frequent than annual
r = nominal rate, m = compounding periods/year
Continuous Compounding FV = PV × eʳⁿ When m → ∞ (continuous compounding)
📅 2. Annuity Formulas (Equal Payments Every Period)
Formula NameFormulaUse When
PVIFA
PV Interest Factor for Annuity		 PVIFA(i,n) = [1 − (1+i)⁻ⁿ] / i Find PV of equal annual receipts/payments
EMI, loan repayment, welfare fund single deposit
FVIFA
FV Interest Factor for Annuity		 FVIFA(i,n) = [(1+i)ⁿ − 1] / i Find future total of equal annual deposits (sinking fund)
PV of Annuity PV = A × PVIFA(i,n) Single deposit needed now to fund equal future payments
FV of Annuity FV = A × FVIFA(i,n) Total accumulated from equal periodic deposits
EMI / EAW / Capital Recovery A = PV × [i(1+i)ⁿ / ((1+i)ⁿ−1)]
A = PV / PVIFA(i,n)		 Find equal annual payment for a given loan/PV
Car loan, mortgage, EAW comparison
Sinking Fund Factor A = FV × [i / ((1+i)ⁿ−1)]
A = FV / FVIFA(i,n)		 Annual deposit needed to accumulate a target FV
Key Trigger: If payments are equal every year → use PVIFA or FVIFA. If amounts differ year to year → use PVIF individually for each year.
📊 3. Project Evaluation (NPV, IRR, PI, PBP)
MethodFormulaDecision Rule
NPV
Net Present Value		 NPV = Σ[CFₜ / (1+i)ᵗ] − I₀
NPV = Σ[CFₜ × PVIF(i,t)] − I₀
I₀ = Initial investment, CFₜ = Cash flow at year t		 NPV > 0 → Accept ✅
NPV < 0 → Reject ❌
Higher NPV → Better project
IRR
Internal Rate of Return		 Find i* where NPV = 0
IRR ≈ i₁ + [NPV₁/(NPV₁−NPV₂)] × (i₂−i₁)
Interpolate between two trial rates		 IRR > MARR → Accept ✅
IRR < MARR → Reject ❌
PI / BCR
Profitability Index / Benefit-Cost Ratio		 PI = PV of Inflows / Initial Investment
PI = (NPV + I₀) / I₀		 PI > 1 → Accept ✅
PI < 1 → Reject ❌
Higher PI → Better (when comparing)
PBP
Payback Period		 Sum cash flows until = I₀
No discounting; cumulative addition year by year		 Shorter PBP → Preferred
Compare with target payback period
ARR
Accounting Rate of Return		 ARR = Avg Annual Profit / Avg Investment × 100
Avg Investment = (I₀ + Salvage) / 2		 ARR > target rate → Accept ✅
No time value of money used
MARR
Minimum Attractive Rate of Return		 Compare: IRR vs MARR
MARR is the minimum acceptable return (given in problem)		 It's a threshold, not a formula. IRR must exceed it.
⚖️ 4. Present Worth & Future Worth (Comparing Alternatives)
MethodFormulaUse When
Present Worth (PW) PW = −C₀ + Annual Benefits × PVIFA(i,n) + Salvage × PVIF(i,n) Compare two cost alternatives; pick lower cost (or higher PW of benefits)
Future Worth (FW) FW = PW × (1+i)ⁿ Compare at end of project life instead of beginning
EAW / EAC
Equal Annual Worth / Cost		 EAW = Annual Revenue − C₀/PVIFA + Salvage/FVIFA
EAW = Annual Revenue − C₀ × CRF + Salvage × SFF		 Compare alternatives with different lives; pick highest EAW (or lowest EAC)
Capital Recovery Factor (CRF) CRF = i(1+i)ⁿ / ((1+i)ⁿ − 1) = 1/PVIFA(i,n) Annual cost of owning an asset (depreciation + interest)
Aqueduct-type questions: Calculate PW of each option → Pick the one with lower PW of cost (or higher PW of net benefit).
📉 5. Break-Even Analysis
FormulaFormula ExpressionNotes
Contribution per Unit C = SP − VC
SP = Selling Price/unit, VC = Variable Cost/unit		 Variable Cost = Variable Mfg + Variable Selling costs
P/V Ratio (C/S Ratio) P/V = (SP − VC) / SP × 100
P/V = Contribution / Sales × 100		 Measures % of sales that covers fixed costs; higher = better
BEP in Units BEP (units) = Total Fixed Cost / (SP − VC)
BEP (units) = FC / Contribution per unit		 FC = Fixed Factory OH + Fixed Selling OH
BEP in Rupees (Sales) BEP (Rs.) = Total FC / P/V Ratio
BEP (Rs.) = BEP (units) × SP		 Sales value at no-profit no-loss point
Units for Target Profit Units = (FC + Target Profit) / Contribution per unit "Units needed to earn Rs. X profit"
Sales for Target Profit Sales = (FC + Target Profit) / P/V Ratio Revenue needed to achieve target profit
Margin of Safety (MOS) MOS = Actual Sales − BEP Sales
MOS % = (MOS / Actual Sales) × 100		 How far sales can drop before loss occurs
Profit Profit = MOS × P/V Ratio
Profit = (Sales − BEP Sales) × P/V Ratio		 Quick profit calculation using MOS
🏭 6. Depreciation Methods
MethodFormulaNotes
Straight Line (SLM) D = (C − S) / n
Book Value = C − D×t		 C = Cost, S = Salvage value, n = life in years
Equal depreciation every year
Written Down Value (WDV) / Declining Balance D_t = BV_{t-1} × d
d = 1 − (S/C)^(1/n)		 d = depreciation rate; BV reduces each year
Higher depreciation in early years
Sum of Years Digits (SYD) D_t = (n−t+1) / SYD × (C−S)
SYD = n(n+1)/2		 Accelerated depreciation; weights earlier years more
Double Declining Balance (DDB) D_t = (2/n) × BV_{t-1} Double the SLM rate applied to book value
Units of Production D = (C−S) / Total Units × Units Produced Based on actual usage, not time
🔧 7. Cost Estimation & Indices
ModelFormulaUse When
Power-Sizing (Six-Tenths) Model C₂ = C₁ × (Q₂/Q₁)ˣ × (Index₂/Index₁)
x = scaling exponent (often 0.6 or 0.75 in problems)		 Estimating cost of equipment with different capacity
Centrifuge, reactor, plant sizing questions
Cost Index Method C₂ = C₁ × (Index₂ / Index₁) Adjusting past cost to present using price index
Factor Method Total Cost = f × C_major equipment
f = installation/indirect cost factor		 Quick plant cost estimation
Make vs Buy Decision Make cost = Variable Cost/unit × Q + Fixed Cost
Buy cost = Purchase Price/unit × Q
Compare at expected quantity Q		 Should firm manufacture or outsource a component?
📈 8. Inflation Adjustments
ConceptFormulaNotes
Real vs Nominal Cash Flow Actual CF = Real CF × (1 + f)ⁿ
f = annual inflation rate		 Convert today's value to future inflated value
Combined Interest-Inflation Rate i_c = i + f + i×f
≈ i + f (for small values)		 Discount rate when cash flows are in nominal terms with inflation
PV with Inflation (Welfare Fund type) Inflate each CF: CF_t = A × (1+f)ᵗ
Then PV = Σ [CF_t × PVIF(i,t)]		 Employee welfare fund with inflation: inflate amounts, then discount at nominal rate
🎲 9. Expected Value & Decision Under Uncertainty
ConceptFormulaUse When
Expected Value E(X) = Σ [P_i × X_i]
P_i = probability, X_i = outcome value		 Dam height optimization, profit with probability, any risk analysis
Expected PW E(PW) = Σ [P_i × PW_i] When different scenarios have different probabilities
Expected Annual Damage E(Damage) = Σ [P(flood) × Damage/year] Dam/flood-protection cost-benefit analysis (dam height problems)
🎯 10. Sensitivity Analysis
ConceptFormulaNotes
Break-Even Value of Variable Set NPV = 0, solve for the variable "By how much can X change before project becomes unviable?"
% Change to Break Even % Change = (Break-even value − Base value) / Base value × 100 Identify which variable (sales volume, price, cost) is most critical
Sensitivity Ratio SR = % change in NPV / % change in variable Higher SR = more sensitive = riskier variable
